# 🚀 Splunk Log Analysis & Monitoring Project

## 📌 Project Overview
This project demonstrates log monitoring and analysis using Splunk. 
It focuses on collecting logs, indexing data, and performing search queries for troubleshooting and monitoring.

## 🏗️ Architecture
- Forwarder → Collects logs from servers
- Indexer → Processes and stores logs
- Search Head → Searches and visualizes data

## 🎯 Objective
- Monitor system logs
- Perform troubleshooting
- Analyze log data using Splunk queries

## ⚙️ Steps Performed
1. Configured Splunk Forwarder
2. Ingested logs to Indexer
3. Performed searches using Search Head
4. Analyzed logs for errors and patterns

## ✅ Outcome
- Successfully monitored logs
- Identified error patterns
- Performed log-based troubleshooting

## 📘 Learning
- Splunk architecture
- Log ingestion & indexing
- SPL query usage
